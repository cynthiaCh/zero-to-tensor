# Zero to Tensor

从数学直觉到 AI 张量的学习实验室。

我发现自己真正学懂数学，往往不是因为看了更多视频或教程，而是因为边抄写边理解。抄写让我慢下来，逐句处理别人的表达，再把它重新变成自己的理解。

这个项目用来记录这个过程：

- 抄下关键句子
- 写出真实困惑
- 捕捉突然通了的瞬间
- 用自己的话重说
- 用 Python 做小实验
- 把数学直觉连接到 AI
