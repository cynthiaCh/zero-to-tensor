# Zero to Tensor

从数学直觉到 AI 张量的学习实验室。

这个项目记录我如何通过抄写、追问、顿悟、代码实验和复盘，一步步理解数学与 AI。它不是一套已经完成的教程，而是一份持续生长的学习档案。

## 在线阅读

GitHub Pages 可以直接使用 `docs/` 目录发布：

- 首页：`docs/index.md`
- 文章：`docs/notes/`
- 图片：`docs/assets/images/`

启用方式：

1. 打开 GitHub 仓库的 `Settings`
2. 进入 `Pages`
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`，目录选择 `/docs`

## 记录方法

每个概念尽量经过六步：

1. 原文抄写：只抄真正卡住我或打动我的句子。
2. 我没懂什么：写下最真实的问题。
3. 我突然通了什么：记录顿悟瞬间。
4. 我用自己的话重说一遍：讲给昨天的自己听。
5. 和 AI 的连接：把数学直觉接到 AI 概念上。
6. 一句话自测：如果一句话讲不清，就继续追问。
